# Object类型

* 创建Object实例的方式一般有两种  
  1. 使用new+构造函数定义  
  2. 使用对象字面量定义

  * ```
    var person = {
        name:'xiaoming',
        age: 2
    }
    ```

    属性值也可以使用字符串“name”,"age“表示

  * 如果直接定义一个{} 那么跟new Object\(\)相同，但是使用对象字面量时，不会调用Object构造函数

* 变量属性的访问

  * 直接使用点语法来进行访问

    * .propertyName需要和对象中定义的属性名一样

  * 使用方括号\[\]进行访问

    * 方括号内可以使用字符串或者使用变量
